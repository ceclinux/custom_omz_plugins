# Configuration

1. Read doc http://sc.ftqq.com/3.version and get your SCKEY
2. put `export SCKEY=YOUR SCKEY HERE` in your `.zshrc` file

# How to use

`fangtang followed by the command you want to execute`

for example

`fangtang du -a`
